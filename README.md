# PayTe - Smart Expense Tracker

**PayTe** is a modern, premium expense tracking dashboard designed to help you manage your finances with ease. This application provides real-time analytics, wallet management, and a seamless user experience with both light and dark modes.

## ✨ Features

- **📊 Interactive Dashboard**: Get a real-time overview of your Total Balance, Income, and Expenses.
- **📈 Advanced Analytics**: Visualize your spending habits with dynamic Charts (Income vs. Expense, Weekly Trends) powered by Chart.js.
- **💳 Wallet Management**: Manage your cards and view a detailed transaction history.
- **🌗 Dark Mode**: Toggle between a clean Light Mode and a sleek Dark Mode for comfortable viewing.
- **📱 Fully Responsive**: Optimized for desktops, tablets, and mobile devices.
- **🔐 Secure Mock Auth**: Includes Login/Signup functionality (Prototype).

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Libraries**: 
  - [Chart.js](https://www.chartjs.org/) (Data Visualization)
  - [FontAwesome](https://fontawesome.com/) (Icons)
- **Deployment**: Vercel

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/expense-tracker.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd expense-tracker
   ```

3. **Run the application**:
   - Simply open `index.html` in your web browser.
   - For the best experience, use a local development server (e.g., Live Server in VS Code).
